# Requisitos e Instalación

Este proyecto requiere Python 3 y algunas librerías externas.

# Instrucciones para ejecutar el programa

1. Obigatorio tener Python instalado (recomendado: versión 3.10 o superior).
   para  verificar la version con el comando:

   python --version

   
2. Ejecutar el programa con el siguiente comando:

   python codigofinal.py

##El programa codigofinal.py puede instalar de forma automatica las librerias y actualizar pip, pero en dado caso no funcione realizar lo siguiente:

# Actualizar pip 

Para asegurar que la instalación de librerías funcione correctamente, es recomendable actualizar pip a la última versión antes de instalar los paquetes

python -m pip install --upgrade pip

# Instala las librerías necesarias. 
   Abrir la terminal en la carpeta del proyecto y ejecutar

   pip install -r requirements.txt

# Notas adicionales
Si se utiliza un sistema operativo como Windows, asegúrarse de tener Python agregado al PATH.

Si ve errores de permisos, intentar ejecutar la terminal como administrador.
